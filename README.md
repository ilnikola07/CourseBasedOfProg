****📋 Описание проекта****


Данный репозиторий содержит программное решение для курсовой работы по предмету "основы программирования". Приложение разработано на языке C# и 
представляет собой игровой проект по теме "текстовый квест на произвольную тему". Основная игровая задача - исходя из описаний пещер, успеть за 
определённое время (10 минут) найти выход из западни.


Проект реализован с использованием архитектуры, разделяющей пользовательский интерфейс и бизнес-логику, что обеспечивает модульность и удобство тестирования.



**🛠 Технологический стек**


Язык программирования: C# версия 12.0


Платформа: .NET Framework 4.8.1


UI Framework: Windows Forms (WinForms)


Архитектура: Class Library (выделение логики в отдельный проект PowerLogic)


Тестирование: MSTest (Unit-тесты)


**📦 Установка и настройка**


Для корректной работы проекта необходимо строго соблюдать структуру каталогов и настройки зависимостей.
1. Получение исходного кода. Склонируйте репозиторий или загрузите архив с исходным кодом.


2. Открытие проекта. Запустите Visual Studio (рекомендуется последняя стабильная версия).
Откройте файл решения (.sln), расположенный в папке проекта. 


3. Начните игру! Ознакомьтесь с предысторией и начните игровой процесс. Удачи...


4. В случае, если у вас не появляются фоны в игровой форме или не появляется текст описания, скопируйте папку и файлы quests.txt, caves_graph.txt, Caves в папку с исполняемым файлом.



**Тестирование**


Проект покрыт модульными тестами с использованием фреймворка MSTest.
Откройте Test Explorer (Тестовый обозреватель) в Visual Studio (Test -> Windows -> Test Explorer).
Нажмите Run All Tests (Запустить все тесты).
Убедитесь, что все тесты пройдены успешно (должно гореть 25 зелёных галочек).
